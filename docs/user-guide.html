Proje Özeti

Çevrimiçi Türkçe Sözlük, Node.js ve Express ile geliştirilen web tabanlı bir sözlük uygulamasıdır. Yönetici kullanıcıların Türkçe kelimeler ve tanımlar ekleyip yönetmesine olanak sağlarken, son kullanıcılara kelime anlamları, eş ve zıt anlamlı kelimeleri hızlıca bulabilme imkânı sunar. Projede, kelimeler, tanımlar ve kelime ilişkileri için çok tablolu ilişkisel veritabanı ve RESTful API yapısı kullanılmıştır.

Anahtar Özellikler:

Özellik Açıklama
Kelime ve Tanım Yönetimi Yöneticiler yeni kelimeler, çoklu tanımlar ve örnek cümleler ekleyebilir.
Eş/Zıt Anlamlılar İlişkili kelimeler saklanarak zengin ve modüler bir veri modeli oluşturulmuştur.
Arama Fonksiyonu Kullanıcılar kelime arayarak tanımlar, örnekler ve ilgili kelimelere erişebilir.
Yönetici Paneli İçerik yönetimi için güvenli ve kullanıcı dostu arayüz.

Klasör Yapısı

Proje, Node.js Express uygulamaları için tipik MVC benzeri bir yapıdadır:


project-root/
├── index.js             # Ana uygulama giriş dosyası
├── routes/              # Route tanımları
│   ├── api.js           # Genel API uçları
│   ├── admin.js         # Yönetici paneli route’ları
│   └── index.js         # Ön yüz sayfalarının route’ları
├── controllers/         # İş mantığını içeren handler’lar
│   ├── wordController.js
│   ├── definitionController.js
│   └── relationController.js
├── models/              # Veritabanı modelleri ve yardımcı modüller
│   ├── db.js            # MySQL bağlantısı (mysql2)
│   └── wordModel.js     # Kelime sorgulama fonksiyonları
├── views/               # Sunucu tarafı render şablonları (şu an boş)
├── public/              # Statik dosyalar
│   ├── index.html
│   ├── admin.html
│   ├── css/
│   ├── js/
│   └── images/
└── ...diğer dosyalar

Kurulum ve Yükleme

Node.js (LTS sürümü önerilir) ve NPM kurulu olmalıdır.

MySQL veritabanı sunucusu çalışır durumda olmalıdır.

.env dosyası veritabanı bilgilerini içerir:


DB_HOST=localhost
DB_USER=root
DB_PASSWORD=mysql_sifreniz
DB_NAME=sozluk_db
ADMIN_PASSWORD=gucluSifre
SESSION_SECRET=oturumAnahtari

Kurulum Adımları:

Adım Komut/Açıklama
1. Depoyu klonlayın: git clone
2. Bağımlılıkları yükleyin: npm install
3. Veritabanını oluşturun ve tabloları ekleyin.
4. .env dosyasını yapılandırın.
5. Sunucuyu başlatın: npm start veya node index.js

Veritabanı Yapısı

Tablo Alanlar
words Kelimeler (id, word)
definitions Tanımlar (id, word_id, definition_text, example_sentence)
relations Kelime ilişkileri (id, left_word_id, right_word_id, relation_type)

Kimlik Doğrulama ve Güvenlik

Yönetici girişi için express-session kullanılır.

Yönetici rotaları checkAdmin middleware ile korunur.

.env dosyasında hassas bilgiler saklanır.

HTTPS kullanımı ve şifreleme önemlidir.

API Uç Noktaları

Herkese Açık:


Kelime Arama: GET /search?kelime={kelime}
Eş Anlamlılar: GET /api/synonyms?word={kelime}
Zıt Anlamlılar (varsa): GET /api/antonyms?word={kelime}

Yönetici:


Yönetici Girişi: POST /admin/login
Kelime Ekleme: POST /admin/words
Tanım Ekleme: POST /admin/definitions
İlişki Ekleme: POST /admin/relations
Kelime Silme: DELETE /admin/words/:id
