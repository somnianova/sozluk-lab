TÜRKÇE SÖZLÜK BURADA!

Bu proje, kullanıcıların kelime tanımları, örnek cümleler, eş anlamlılar, zıt anlamlılar, deyimler, alıntılar gibi zengin sözlük öğelerine erişmesine olanak tanıyan bir Node.js ve MySQL tabanlı web uygulamasıdır. Aynı zamanda, bir Admin Paneli üzerinden kelime ve anlam (definition) yönetimi yapılabilmektedir.

Kullanıcı Arayüzü (Public)

Arama kutusu (AJAX veya fetch ile “ara”/“search” işlevi).
Sözcük başlıkları, tanımlar, örnek cümleler ve köken bilgisi.
Synonym, antonym, idiom vb. alanları dinamik olarak doldurabilme.

Admin Paneli

Kelimeleri listeleyebilme, ekleyebilme, düzenleyebilme.
Not: Kod içerisinde DELETE /admin/words/:id endpoint’i henüz eklenmemiştir. Bu yüzden Readme’deki “kelime silme” özelliği taslak aşamasındadır. İleride eklenmesi planlanmaktadır.
Definitions (anlam) ekleme, silme, düzenleme.
Kelimeler arası ilişki (synonym, antonym, idiom, collocation, quote vb.) tanımlama.
Alfabetik tarama, sayfalama ve sıralama seçenekleri.
Oturum bazlı yetkilendirme (admin şifre kontrolü).

Veritabanı (MySQL)

  • words tablosu: Kelimeye ilişkin temel bilgiler (word, origin, frequency_tags, audio_url).
  • definitions tablosu: Bir kelimenin birden fazla anlamını tutar (definition_text, example_sentence, part_of_speech, sense_label vb.).
  • relations tablosu: Kelimeler arası ilişki (synonym, antonym vb.) yapmak için kullanılır.
  • Hem full-text arama hem de LIKE bazlı arama desteği.

Gereksinimler

Node.js (en az v14 veya üstü), MySQL 5.7+, Tarayıcı (Chrome, Firefox, vb.)

Kurulum

3a Projeyi Klonla veya İndir

git clone https://github.com/kullanici/turkce-sozluk-lab.git
cd turkce-sozluk-lab

3b Bağımlılıkları Yükle

npm install

3c MySQL Veritabanını Oluştur

★ Bir veritabanı (örnek: sozluk_db) oluşturun.
★ words, definitions, relations tablolarını yaratmak için örnek şema veya kendi SQL şemanızı çalıştırın.

3d .env Dosyasını Düzenle

DB_HOST, DB_USER, DB_PASSWORD, DB_NAME, ADMIN_PASSWORD, SESSION_SECRET, PORT gibi alanları doldurun.

1c Veritabanı Düzeni

words
id (INT PK, AUTO INCREMENT)
word (VARCHAR)
origin (TEXT)
frequency_tags (VARCHAR)
audio_url (VARCHAR)

definitions
id (INT PK, AUTO INCREMENT)
word_id (INT FK -> words.id)
definition_text (TEXT)
example_sentence (TEXT)
part_of_speech (VARCHAR)
sense_label (VARCHAR)

relations
id (INT PK, AUTO INCREMENT)
left_word_id (INT FK -> words.id)
right_word_id (INT FK -> words.id)
relation_type (VARCHAR)

Kullanım

1a Çalıştırma

npm start

veya

node index.js

Ardından http://localhost:3000 adresinde uygulamayı görebilirsiniz.

1b API Uç Noktaları

  • GET /search : Full-text arama
  • GET /search-like : LIKE tabanlı arama
  • GET /api/synonyms?word=...
  • GET /api/antonyms?word=...
  • GET /api/idioms?word=...
  • GET /api/quotes?word=...
  • GET /api/collocations?word=...
  • GET /api/word-family?word=...
  • GET /words-by-letter?letter=...

1c Admin Uç Noktaları (Tümü checkAdmin korumalı)

  • GET/POST /admin/login : Admin oturumu açma
  • GET /admin/logout : Oturumu kapatma
  • POST /admin/add-word : Kelime ekleme
  • GET /admin/words : Kelime listeleme (sayfalama/sıralama)
  • GET /admin/words/:id : Tek bir kelimenin detaylarını getirir
  • PUT /admin/words/:id : Kelime bilgilerini günceller
  • DELETE /admin/words/:id : Henüz implement edilmedi, gelecekte eklenecek.
  • GET /admin/definitions : Definition listeleme (sayfalama)
  • PUT /admin/definitions/:id : Bir definition güncelleme
  • DELETE /admin/definitions/:id : Definition silme
  • POST /admin/add-definition : Yeni definition ekleme
  • GET/POST /admin/relations : Kelimeler arası ilişki listeleme/ekleme
  • DELETE /admin/relations/:id : İlişki silme
  • GET /admin/last-words : Son eklenen 10 kelime

Admin Paneli Kullanımı

2a Giriş: /admin/login adresine gidin.
Varsayılan kullanıcı adı: admin, şifre: .env içindeki ADMIN_PASSWORD.

2b Panel Ana Ekran: admin.html üzerinden:
Kelimeler: ekle/listele/güncelle (silme butonu var ama DELETE route henüz eklenmemiştir).
Definitions: ek anlam ekle/listele/sil.
İlişkiler: synonym, antonym, idiom vb.
Alfabetik: harfe göre kelime listeleme.
Yardım: kullanım rehberi.

2c Kelime Düzenleme: admin-edit.html?id=5 gibi bir sayfada ilgili kelimenin adını, kökenini ve tanımlarını görebilir, güncelleyebilirsiniz.

2d Çıkış: /admin/logout ile oturumu kapatabilirsiniz.
